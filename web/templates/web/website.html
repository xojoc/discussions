Display comments on your website

Just include a link to this site, like this: 

    https://discu.eu/?q=URL&submit_title=article title
the submit_title parameter is optional, but when included it will prefill the submission title if the user decides to submit your story to a platform.

If you want to include a link only on pages with relevant dicussions then first call the API (use the discussions_counts method).

The API returns the total count of comments and the number of comments for each platform and subreddit. This way you can include a link only if you're interested in the comments of a specific platform or subreddit.
